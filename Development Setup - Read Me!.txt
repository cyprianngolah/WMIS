Requirements - Visual Studio 2012 or 2013, SQL Management Studio 2012, .NET 4.5

0) Ensure you have nuget installed. http://www.nuget.org/
1) Create an IIS 4.5 Web Site with an App Pool called YEC and ensure that the path is to the YEC folder beside this file. Setup host name bindings for host name of "YEC"
1a) For Windows 8 Dev - the IIS will use Application Pool Identity, so use the user "IIS AppPool\YEC" for step 2
1b) For Windows 7 Dev - the IIS will use Network Service by default, so use "Network Service" for step 2
2) In the YEC folder beside this document, select properties and give the user specified in 1a/1b Modify permissions
3) Add an entry to hosts file (C:\Windows\System32\Drivers\etc\hosts):
	127.0.0.1			yec
4) Add the following Registry Information:
	Navigate To:
		HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services\LanmanServer\Parameters
	Add:
		Value name: DisableStrictNameChecking
		Data type: REG_DWORD
		Radix: Decimal
		Value: 1 
5) Follow the steps for Method 1 at http://support.microsoft.com/kb/896861 
6) In IIS Manager for YEC site, click Authentication and ensure only Windows Authentication is checked
7) Compile the sln
8) Navigate to http://yec
